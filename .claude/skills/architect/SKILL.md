# Skill Architect

The Skill Architect is a meta-skill designed to guide creating, modifying, and maintaining high-quality, professional-grade skills.

## Goal

Design skills that trigger reliably and guide LLMs through structured workflows.

## Rules

- Descriptions must include trigger keywords and use-case specifics
- Keep skills under 200 lines when possible
- Use "Why" explanations, not imperatives
- One responsibility per skill
- Proper frontmatter: name, description
