# Pages (`frontend/src/pages/`)

このディレクトリには、React Routerのルートに対応するページレベルのコンポーネントが含まれています。

## [`GamePage.jsx`](./GamePage.jsx)
ゲームの詳細を表示するメインページです (`/games/:slug`)。

### 主な機能
1. **データ取得**: URLパラメータ (`slug`) を基に、API (`/api/games/:slug`) からゲームデータを取得します。
2. **SEO対策**: `react-helmet-async` を使用して、動的に `<title>`, `<meta description>`, OGPタグ、構造化データ (`ld+json`) をヘッドに注入します。
3. **動的レンダリング**:
    - **Markdown描画**: ルールがテキスト形式の場合、`react-markdown` で整形して表示します。
    - **構造化データ描画**: ルールがJSONオブジェクトの場合、セクションごとに展開して表示します。
4. **インタラクション**: 編集モーダルの開閉、再生成ボタンのハンドリング、シェア機能などを統括します。

## [`DataPage.jsx`](./DataPage.jsx)
（存在する場合）統計情報やデータ一覧を表示する管理用または分析用のページです。

## 設計パターン
- ページコンポーネントは、ルーティングのエントリーポイントとして機能し、必要なデータをフェッチして子コンポーネント（`components/`）に渡す「Container Component」としての役割を担います。
