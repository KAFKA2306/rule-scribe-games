# Utils (`app/utils/`)

アプリケーション全体で使用される汎用的なユーティリティ関数群です。特定のビジネスロジックに依存しない、純粋な関数を配置します。

## モジュール一覧

### [`affiliate.py`](./affiliate.py)
- **役割**: アフィリエイトリンクや検索URLの生成。
- **主要関数**:
    - `amazon_search_url(query)`: ゲームタイトルを基に、Amazonの検索結果URLを生成します（アフィリエイトタグ付与のロジックを含む場合があります）。

### [`slugify.py`](./slugify.py)
- **役割**: URLフレンドリーなスラッグ（Slug）の生成。
- **ロジック**: 日本語タイトルを含む任意の文字列を、URLとして安全な形式（英数字とハイフンのみなど）に変換したり、あるいはそのままエンコードして使用可能にしたりするための正規化処理を行います。

### [`logger.py`](./logger.py)
- **役割**: (Deprecated/Moved) ロガー設定は `app/core/logger.py` に移動しましたが、互換性や特定のユーティリティ用途で残されている場合があります。基本的には `app/core` のものを使用してください。

## 設計方針
- **Stateless**: ここに含まれる関数は副作用を持たず、入力に対して常に同じ出力を返す純粋関数であることを目指します。
