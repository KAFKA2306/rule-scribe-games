# Note.com 戦略と運用ガイド

このドキュメントは、コンテンツ配信プラットフォーム **Note.com** における「ボドゲのミカタ」の運用戦略、プロフィール設定、および記事品質の基準を定義します。

## 1. プロフィール設定 (Branding & SEO)

プロフィールは、読者が「フォローするかどうか」を決める最大の要素です。以下の設定を維持してください。

### 名前 (Name)
検索性と信頼性を両立させる命名規則です。
*   **設定値**: `ボドゲのミカタ｜初心者向けルール解説`
*   **戦略**:「ブランド名」＋「何を提供する人か（ベネフィット）」
    *   `ボドゲのミカタ`: 覚えやすいブランド名。
    *   `初心者向け`: ターゲット層を明確化。
    *   `ルール解説`: 検索されやすいキーワードを含める。

### 自己紹介 (Bio) - 140文字制限
限られた文字数で「権威性」「解決策」「誘導(CTA)」を提示します。

*   **現在の設定値 (135文字/140文字)**:
    ```text
    初心者向けボドゲ用語・ルール解説『ボドゲのミカタ』公式。
    「説明書が難しい」「勝ち方がわからない」を0秒で解決！
    「30秒要約」と「コツ」であなたのボドゲライフをサポートします。
    👇解説＆検索
    https://rule-scribe-games.vercel.app/
    ```
*   **構成要素**:
    1.  **Who**: 『ボドゲのミカタ』公式。
    2.  **Pain**: 説明書が難しい、勝ち方がわからない。
    3.  **Solution**: 0秒で解決、30秒要約。
    4.  **CTA**: 解説＆検索リンク。

### 画像 (Assets)
*   **ヘッダー画像**: `assets/02_ボドゲのミカタ.jpg` (ブランドロゴ/コンセプト)
*   **アイコン画像**: `assets/01_探検家ミープル君.png` (親しみやすいマスコット)

## 2. 記事コンテンツ品質 (Quality Control)

AI特有の「ロボットっぽさ」を排除し、**「中の人（ボドゲ好き）」の熱量**を感じさせる記事を目指します。

### 見出し (Headers)
*   **NG**: `なぜ今、面白いのか？ (3 Points)` (英語の補足や括弧書きはAIっぽい)
*   **OK**: `なぜ今、これが面白いのか？` (シンプルで人間らしい)
*   **フォーマット**: Noteのエディタ仕様に合わせ、必ず `<h3>` タグを使用します。

### リンク埋め込み (Embedded Cards)
単なるテキストリンクはクリック率が低いため、**必ずブログカード形式**で埋め込みます。

*   **Amazonリンク**: アフィリエイトタグ `bodogenomikat-22` を必須とします。
*   **内部リンク**: 自社サイト `rule-scribe-games.vercel.app` への誘導を強化します。
*   **技術的な注意**: 
    Amazonリンクのカード化は不安定なため、**「Space -> Backspace -> Enter」** のシーケンス入力を用いて確実にカード生成をトリガーします。（Skill: `write-article.md` 参照）

## 3. SEO / 検索流入戦略

### ターゲットキーワード
*   **ビッグワード**: `ボードゲーム`, `アナログゲーム`, `ルール`, `遊び方`
*   **トレンドワード**: Twitter(X)で話題の新作、テレビで紹介されたゲーム
*   **ニッチ**: `[ゲーム名] 勝ち方`, `[ゲーム名] コツ`

### 外部連携
*   **記事内リンク**: 常に自社サイトの「ルール詳細ページ」へリンクし、Noteを入り口としてWebサイトへの送客を図ります。
*   **ハッシュタグ**: 適切なタグ（例: `#ボードゲーム` `#Splendor`）を5〜6個設定し、Note内検索での露出を増やします。
