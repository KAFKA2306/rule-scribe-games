KEY FACTS
================================================================================

Root Cause: Connection Pool Exhaustion
  • 50 games × 4 operations/game = 200 async tasks
  • Python thread pool: only 8 workers
  • Supabase pool: only 10-20 connections
  • By game 15: all connections exhausted → timeouts

Why game 15-20? 
  • 15 games × 4 ops = 60 queued operations
  • 8 threads × 5 sec each = 40 seconds delay
  • By game 15-20, wait time exceeds timeout (30s) → ERROR

Fix: Limit concurrency with semaphore
  • Max 5 operations at a time (not unlimited)
  • Queue operations instead of spawning threads
  • Connections reused efficiently
